A simple Unity Editor tool to automatically Add, Commit, and Push changes to GitHub with a single click (or shortcut).

## Features
- **Auto Add & Commit:** Automatically stages all changes and commits with a timestamp.
- **Auto Push:** Pushes changes to the current branch (creating it if it doesn't exist).
- **Shortcut:** Use `Ctrl + G` to backup instantly.

### Method 1: Easy Install (.unitypackage)
1. Download the `OneClickGit.unitypackage` file from this repository.
2. Open your Unity project.
3. Double-click the downloaded file to import it.

### Method 2: Manual Install
1. Create a folder named `Editor` inside your `Assets` folder (if you don't have one).
2. Download `GitAutomation.cs`.
3. Place the script inside the `Editor` folder.

## How to Use
1. Make sure Git is installed on your OS.
2. Open Unity.
3. Go to `Git Tools` > `Backup and Push` in the top menu.
4. Or simply press **Ctrl + G**.
